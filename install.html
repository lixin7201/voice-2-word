安装大宜宾录音助手

不需要谷歌商店，也不需要 VPN。下载启动器后，每次打开都会自动更新录音助手插件。

下载录音助手浏览器启动器 下载后解压。Mac 用 Chrome 就双击 open-voice-to-word-chrome.command；用 Atlas 就双击 open-voice-to-word-atlas.command。
  1. 下载 zip 后先解压。
  2. Mac 用 Chrome：双击 open-voice-to-word-chrome.command。
  3. Mac 用 ChatGPT Atlas：双击 open-voice-to-word-atlas.command。
  4. Windows 用 Chrome：双击 open-voice-to-word-chrome-windows.cmd。
  5. 启动器会自动更新插件并打开专用浏览器窗口。
这种方式不会把插件塞进你平时那个浏览器窗口；请从这个启动器进入录音网页。Atlas 如果不支持加载插件，请改用 Chrome 启动器。
