安装大宜宾录音助手

不需要谷歌商店，也不需要 VPN。下载最新版插件压缩包，解压后在 Chrome 里加载即可。

下载最新版插件压缩包 当前版本 0.1.2。下载后解压，再到 Chrome 扩展管理页加载解压后的文件夹。
  1. 下载 zip 后先解压到一个固定文件夹。
  2. 打开 Chrome 的 chrome://extensions。
  3. 打开右上角“开发者模式”。
  4. 点“加载已解压的扩展程序”，选择刚才解压出来的文件夹。
  5. 以后有新版时，点插件里的更新提示下载新版 zip，解压后覆盖旧文件夹，再在扩展管理页点“重新加载”。
浏览器不能自动把 zip 下载到旧插件目录，也不能自动解压覆盖。这一步需要人工解压覆盖，或者重新选择新文件夹加载。
