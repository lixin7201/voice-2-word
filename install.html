1 第一次安装插件下载、解压、加载到 Chrome，一步一步来。 ›

安装前确认

  • 电脑已经安装 Chrome 浏览器。
  • 能打开本教程页，说明基本连上办公室服务。
  • 已经知道自己的花名/工号；默认密码是 dayibin。
下载并解压
  1. 点击本页上方“下载最新版插件压缩包”。
  2. 下载后找到 voice-to-word-extension-0.1.9.zip 这类文件。
  3. Windows 右键选择“全部解压缩”；Mac 双击 zip。
  4. 把解压出的文件夹放到固定位置，不要放在微信临时目录。
加载到 Chrome
  1. Chrome 地址栏输入 chrome://extensions，按回车。
  2. 打开右上角“开发者模式”。
  3. 点击“加载已解压的扩展程序”。
  4. 选择刚才解压出来的插件文件夹。
  5. 看到“大宜宾录音助手”卡片，就安装成功。
重要：以后不要删除或移动这个解压文件夹。Chrome 加载的是这个文件夹，不是 zip。
2 登录插件普通账号只填花名/工号和密码，不用配置参数。 ›

打开和登录

  1. 点击 Chrome 右上角拼图图标，找到“大宜宾录音助手”。
  2. 点击插件，浏览器右侧会打开侧边栏。
  3. “服务地址”一般不用改。
  4. “工号/花名”输入自己的花名或工号。
  5. “密码”输入 dayibin。如果管理员给过新密码，就用新密码。
  6. 点击“登录”。
建议登录后先把插件固定到右上角：点拼图图标，再点“大宜宾录音助手”旁边的小图钉。
3 认识首页首页能看最近记录，也能进入上传和历史。 ›

首页常用入口

  • “上传录音文件”：手里有 mp3、m4a、wav 等文件时点这里。
  • “查看历史记录”：找以前处理过的录音。
  • “最近历史”：可以直接点某一条查看结果。
  • 右上角头像/名字：进入个人资料、修改密码、退出登录。
4 上传录音文件最稳的方式：先有录音文件，再上传生成纪要或跟单。 ›

上传步骤

  1. 点“上传录音文件”或顶部/侧边“上传”。
  2. 点“选择文件”，选择录音或视频文件。
  3. 填写“录音标题”，例如“6月客户电话沟通”。不填也可以，系统会先用文件名。
  4. 选择处理方式：会议记录选“会议纪要”；客户沟通、商机整理选“跟单信息”。
  5. 点击“上传并生成纪要”。

支持 mp3、m4a、wav、aac、flac、ogg、opus、mp4、mov、webm。长录音会慢一些，上传后可以先离开，稍后到历史里查看。

处理方式怎么选

  • 会议纪要：适合会议、培训、复盘、讨论、电话会议。
  • 跟单信息：适合客户跟进、招聘客户、红娘客户、商机沟通。
  • 不确定时先选会议纪要，后面也可以重新生成。
5 监听网页录音录音在网页播放器里，不方便下载时用这个。 ›

先在网页里播放，再点监听

  1. 先打开保存录音的原网页，并确认自己已登录那个网页账号。
  2. 找到要识别的那条录音，先点播放。
  3. 打开“大宜宾录音助手”，点“监听”。
  4. 点“识别当前页录音”。
  5. 如果出现候选录音，填标题、选处理方式，再点“生成纪要”或“生成跟单”。

看到候选录音后

  • 标记“当前录音”的通常就是最可能的一条。
  • 标题可以改成人能看懂的名字。
  • 如果选的是“跟单信息”，先选择跟单类型，再在候选卡片里填写用户 ID。
  • 点“生成跟单”后可继续下一通电话，后台会继续转写和生成跟单。
  • 如果选的是“会议纪要”，点“生成纪要”后进入详情页等待结果。
  • 如果提示“需手动上传”，说明网页不允许直接读取，下载录音文件后改用上传。
6 查看结果、复制和下载历史、总结、逐字稿、跟单、备注、导出都在详情页。 ›

从历史打开记录

  1. 点“历史”。
  2. 找到录音标题，右侧显示“完成”说明可以看结果。
  3. 搜索框可以输入用户 ID、录音标题、跟单关键词、会议纪要关键词或逐字稿关键词。
  4. 点那条记录进入详情页。
  5. 如果仍是“转写中/总结中”，点“刷新”或稍后再看。
会议概览

会议纪要记录默认打开这里，只放会议总结、要点、待办和风险。

逐字稿

完整文字记录，带时间点和说话人。历史搜索框也能按逐字稿关键词找到录音。

跟单信息

跟单记录默认打开这里，显示用户 ID、跟单类型、阶段、标签和字段化跟单内容。

备注

只写人工补充说明，不再混放 AI 生成的跟单结果。

下载

需要留档时，可下载 Markdown、TXT、DOCX、PDF 等文件。

展开看“总结”截图和步骤
  1. 会议纪要记录进入详情后，默认先看“会议概览”。
  2. 可以点“复制”，把总结粘贴到飞书。
  3. 如果总结不完整，点“重新生成”。
展开看“逐字稿”截图和步骤
  1. 点“会议逐字稿”。
  2. 上方可以播放录音。
  3. 用“搜索逐字稿”查人名、客户名、关键词。
  4. 需要核对原话时，看每段前面的时间点。
展开看“跟单信息/备注/导出”截图和步骤

跟单信息

  • 跟单记录打开后默认进入“跟单信息”。
  • 用户 ID 会优先来自监听候选卡片，也可以在这里补改。
  • 检查阶段、标签、企业/客户、状态和跟单内容。
  • 有需要就修改，最后点“保存跟单修改”。
  • 回到“历史”后，可在搜索框输入用户 ID、跟单正文或跟单字段关键词找回录音。

备注和导出

  • 备注：写自己的补充说明，点“保存备注”。
  • 导出：选择 Markdown、TXT、DOCX 或 PDF 下载。
  • 只要是普通留档，优先下载 DOCX 或 PDF。
7 分享链接把录音、逐字稿、总结按需分享给手机或同事查看。 ›

新版分享入口

  1. 打开一条已处理完成的录音详情。
  2. 点击标题下方的“分享”。
  3. 右侧会弹出分享选择框，不需要往页面下方找。
  4. 勾选要分享的内容：录音、逐字稿、总结。
  5. 选择有效期，默认 7 天，然后点击“复制链接”。

已有链接怎么处理

选择框下方会显示已创建的链接，可以再次复制，也可以撤销不想继续开放的链接。

分享内容
  1. 勾选“录音”，对方可以播放音频。
  2. 勾选“逐字稿”，对方可以查看转写文本。
  3. 勾选“总结”，对方可以查看 AI 总结。
  4. 没勾选的内容，对方不会看到。
手机查看
  1. 把链接发给同事或发到自己手机。
  2. 手机需要连接办公室同一个 Wi-Fi。
  3. 打开链接后可播放录音、查看逐字稿和总结。
  4. 链接过期或撤销后，别人就打不开。
分享链接无需登录也能访问，所以只发给需要查看的人。只勾选“录音”时，对方不会看到逐字稿和总结；只勾选“总结”时，对方不会听到录音。
8 个人资料和修改密码头像、名字、AI 偏好、密码都在右上角个人资料里。 ›

修改个人资料

  1. 点击右上角头像/名字。
  2. 可以修改花名、头像底色、自我介绍。
  3. “AI 生成偏好”可以写你希望纪要更关注什么，比如待办、风险、责任人。
  4. 点“保存个人资料”。

修改密码

  1. 输入旧密码。第一次通常是 dayibin。
  2. 输入新密码，再确认一遍。
  3. 点“保存密码”。
9 以后有新版怎么更新下载新版 zip、解压、刷新扩展。 ›
看到新版提示

插件里如果提示“发现新版插件”，点“下载新版插件包”。也可以直接回到本教程页下载。

解压新版

把新版 zip 解压。可以覆盖原来的旧插件文件夹，也可以放成一个新的固定文件夹。

刷新 Chrome 扩展

打开 chrome://extensions，找到“大宜宾录音助手”，点“重新加载”。

如果放到新文件夹

重新点“加载已解压的扩展程序”，选择新文件夹。

Chrome 不会自动解压 zip，也不会自动覆盖旧插件文件夹。下载新版后，仍然要人工解压并刷新扩展。
10 常见问题哪里卡住，就展开这一栏查。 ›
找不到“加载已解压的扩展程序”

先确认 chrome://extensions 右上角“开发者模式”已经打开。

提示“清单文件缺失”

说明选错文件夹，或者还没解压 zip。请选择里面能看到 manifest.json 的文件夹。

登录失败

先检查花名/工号和密码。默认密码是 dayibin；如果改过密码，就用新密码。

插件打不开

检查解压出来的插件文件夹是否被删除、改名或移动。找不到就重新下载、解压、加载。

网页录音识别不到

先在原网页播放录音，再点“识别当前页录音”。还不行就下载录音后手动上传。

一直处理中

长录音会慢。过一会儿到历史点“刷新”；等很久还没好，把录音标题和时间发给管理员。
